| 字母 | 数量 | 大小写 | 含义                          | 示例（以 2024-03-14 15:30:45 周六 为例） |
|------|------|--------|-------------------------------|------------------------------------------|
| Y/y  | 1    | Y      | 年（4位，带纪元，兼容跨年）   | Y → 2024                                 |
|      | 2    | y      | 年（2位）                     | yy → 24                                  |
|      | 4    | Y      | 年（4位，标准）               | YYYY → 2024                              |
| M/m  | 1    | M      | 月（数字，不带前导零）        | M → 3                                    |
|      | 2    | M      | 月（数字，带前导零）          | MM → 03                                  |
|      | 3    | M      | 月（英文缩写）                | MMM → Mar                                |
|      | 4    | M      | 月（英文全称）                | MMMM → March                             |
|      | 1/2  | m      | 分钟（1位/2位，带前导零）     | m → 30 / mm → 30                         |
| D/d  | 1    | D      | 日（月份中的第几天，1位）     | D → 14                                   |
|      | 2    | D      | 日（月份中的第几天，2位）     | DD → 14                                  |
|      | 1    | d      | 星期（数字，1=周日/7=周六）   | d → 6                                    |
|      | 3    | d      | 星期（英文缩写）              | ddd → Sat                                |
|      | 4    | d      | 星期（英文全称）              | dddd → Saturday                          |
| H/h  | 1    | H      | 小时（24小时制，1位）         | H → 15                                   |
|      | 2    | H      | 小时（24小时制，2位）         | HH → 15                                  |
|      | 1    | h      | 小时（12小时制，1位）         | h → 3                                    |
|      | 2    | h      | 小时（12小时制，2位）         | hh → 03                                  |
| s    | 1    | s      | 秒（1位）                     | s → 45                                   |
|      | 2    | s      | 秒（2位）                     | ss → 45                                  |
| S    | 1-3  | S      | 毫秒（1-3位）                 | S → 5 / SS → 05 / SSS → 005               |
| A/a  | 1    | A      | 上午/下午（大写）             | A → PM                                   |
|      | 1    | a      | 上午/下午（小写）             | a → pm                                   |
